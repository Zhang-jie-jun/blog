# network

&emsp;&emsp;计算机网络是指将地理位置不同的具有独立功能的多台计算机及其外部设备，通过通信线路连接起来，在网络操作系统、网络管理软件及网络通信协议的管理和协调下，实现资源共享和信息传递的计算机系统。

**学习目标：**
+ 理解网络分层模型（OSI七层、TCP/IP四层）
+ 掌握TCP/IP协议族的核心原理
+ 理解HTTP/HTTPS协议的工作机制
+ 掌握RPC框架的设计与实现
+ 具备网络故障诊断与排查能力

## 核心概念

### 网络分层模型
网络协议按照功能划分为不同层次，每层独立完成特定功能，通过接口与相邻层交互。

### TCP/IP协议族
互联网的核心协议族，包括IP、TCP、UDP、HTTP等协议，构成了现代网络通信的基础。

### HTTP/HTTPS
应用层协议，用于Web通信，HTTPS通过SSL/TLS提供安全加密。

### RPC
远程过程调用协议，允许程序像调用本地函数一样调用远程服务。
