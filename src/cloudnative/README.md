# 云原生与虚拟化技术

本目录包含云原生和虚拟化技术相关的教程和实践文档，涵盖容器化、编排、虚拟化等核心技术栈。

| 文档 | 说明 |

## 学习路径

1. **基础入门**：从 Docker 和 QEMU 开始，了解容器和虚拟化基础
2. **深入理解**：学习 KVM、libvirt 和 QCOW2 的原理和实现
3. **进阶学习**：掌握 Kubernetes 编排和 Ceph 分布式存储
4. **实践应用**：结合 OpenStack 或 VMware 构建完整的云平台
